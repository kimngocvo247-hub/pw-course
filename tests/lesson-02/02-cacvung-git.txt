Trong một folder mới được tạo ra, mình thực hiện lần lượt các lệnh sau. Hãy liệt kê các
file theo từng vùng sau khi thao tác một loạt các lệnh sau (sẽ rất tuyệt nếu bạn không
chạy lệnh trên máy thật, mà bạn suy nghĩ trong đầu để ra được kết quả; bạn sẽ nhớ rất
lâu đấy ^^):
a. Tạo 3 file: file1, file2, file3
b. Chạy lệnh: git init
c. Chạy lệnh: git commit -m”init project”
d. Chạy lệnh: git add file1
e. Chạy lệnh: git commit -m”add file1”
f. Chạy lệnh: git commit -m”add file”
3. Push bài tập lên GitHub.
============================================

Answer:
a. Tạo 3 file: file1, file2, file3 => Three files stay in the local area.
b. Chạy lệnh: git init => Three files stay in the working directory area.
c. Chạy lệnh: git commit -m”init project” => Not sure what it would show, perhaps it shows an error.  
d. Chạy lệnh: git add file1 => file1 moves from the working directory are to the staging area
e. Chạy lệnh: git commit -m”add file1” => file1 moves from the staging area to the repository area.
f. Chạy lệnh: git commit -m”add file” => show error because we don't have this file.
